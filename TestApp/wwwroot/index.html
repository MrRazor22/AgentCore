Welcome to AgentCore

This showcase demonstrates composable memory, tool execution with human-in-the-loop approvals, and real-time observability telemetry built directly on the AgentCore framework.

🌦️ Standard Tool Use Ask weather details to run weather tools.
🛡️ Human in the Loop Run a cmd tool requiring user approval.
❌ Safety Rejection Verify that blacklisted commands are denied.

Pipeline Code Configuration

IAgent agent = LLMAgent.Create("chat")
    .WithProvider(provider, new() { ContextWindow = 8000 })
    .WithTools(registry)
    .WithLoggerFactory(loggerFactory)
    // Composable Memory Decorators
    .AddMemoryLayer(inner => new DurableFileMemory(
        new ObservabilityMemoryDecorator(inner, eventBus, sessionId), 
        sessionId))
    // Composable Tool Approvals & Logging
    .AddToolExecutorLayer(inner => new ApprovalToolExecutor(
        new ObservabilityToolExecutorDecorator(inner, eventBus, sessionId), 
        registry, 
        approvalService, 
        sessionId))
    // Composable LLM Logs
    .AddLlmExecutorLayer(inner => new ObservabilityLlmExecutorDecorator(
        inner, 
        eventBus, 
        sessionId))
    .Build();
